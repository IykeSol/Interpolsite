# Local Images

Drop your photos here with **exactly** these filenames:

| File           | Used on                    | Recommended style                                                                                               |
| -------------- | -------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `hero.jpg`     | Home page hero background  | Dark, cinematic — law enforcement, digital investigation, night-time cityscape. Landscape (1920×1080 or wider). |
| `justice.jpg`  | Home page "About" section  | Professional, credible — justice scales, courthouse, legal setting. Portrait or square (800×900).               |
| `security.jpg` | About page mission section | Digital security, blue tech, data forensics aesthetic. Landscape (1200×800).                                    |

> **If a file is missing**, the image area simply shows a dark background — no broken-image icon will appear.

Recommended formats: `.jpg` or `.webp`. Keep files under 500 KB for performance.
